# Spotify Clone

A simple Spotify-inspired web music player built using **HTML, CSS, and JavaScript**.  
This project replicates the core functionality of a music streaming interface such as playing songs, navigating tracks, and displaying album covers.



## Features

- Play and pause songs
- Next and previous track navigation
- Interactive progress bar for tracking song progress
- Dynamic song title and cover display
- Multiple songs with album covers
- Simple and clean Spotify-like interface


## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome (for icons)
